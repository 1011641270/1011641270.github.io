category: 技术
comments: true

```java
String string = "abcd";
  int len = string.length();
  int sum = 0;
  for (int i = 0; i < len; i++) {
   char c = string.charAt(len - i - 1);
   int n = Character.digit(c, 16);
   sum = sum + n*(1<<(4*i));
  }
  System.out.println(sum); //43981
  System.out.println(Integer.parseInt("abcd", 16)); //43981
```java
